# Boreal Desktop

This is the public-safe overview of Boreal Desktop.

Detailed implementation notes, local-runtime policy, and internal rollout planning live only in the local ignored `private/` workspace.

## Public Product Truth

Boreal Desktop is a private, owner-run execution path in progress.

It is intended to let an owner connect a local machine to Boreal so qualifying work can be accepted and completed through the same request lifecycle.

## What Matters Publicly

- the web product remains the front door
- desktop is not public market supply
- desktop is not the main launch wedge
- the request thread remains the system of record

## Boundary

Do not describe Boreal Desktop publicly as a fully shipped open execution network.

This file exists only to keep the public boundary clear.
