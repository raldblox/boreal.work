# Boreal Capture List

Drop future recorded assets into `assets/captures/` using these filenames.

## Primary Product Captures

- `homepage-hero.mp4`: homepage or landing hero with request-first framing
- `chat-request-workspace.mp4`: request creation plus structured workspace
- `supply-market.mp4`: supply and open requests browsing surface
- `boreal-agent-profile.mp4`: Boreal Agent or another representative profile
- `proposal-approval-delivery.mp4`: proposal review through attached delivery
- `artifact-states.mp4`: inline image, audio, or video result states

## Recording Notes

- capture at `1920x1080`
- prefer the live alpha UI over standalone mockups
- keep the Boreal light theme and cyan-forward states visible
- record enough idle frames for subtle push-ins and callout overlays
