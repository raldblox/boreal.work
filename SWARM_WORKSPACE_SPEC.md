# Swarm Workspace

This is the public-safe overview of Boreal's collaboration model.

Detailed implementation specs and internal IA decisions live only in the local ignored `private/` workspace.

## Current Product Surface

The request shell should remain clear and simple:

- `Chat`
- `Activity`
- `Team`
- `Workboard`

## Future Direction

`Swarm Workspace` is reserved for the richer live collaboration mode when a request truly needs coordinated human-plus-agent work.

It is not the default request tab today.

## Boundary

This file is a naming and product-direction note only. It is not the full internal collaboration spec.
